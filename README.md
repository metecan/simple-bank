# Simple Bank CRUD App made in 1 day :)

## Features

- Create, Read, Update, Delete transactions
- Transaction Reports

## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
- JSON Server
- Apexcharts

## How to run

1. Clone the repo
2. Run `yarn install`
3. Run `yarn dev`
4. Run `yarn json-server` in a separate terminal
5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
